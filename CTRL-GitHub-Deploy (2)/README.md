# CTRL - Your Life OS

A personal life operating system to help young adults manage adulting.

## Deploy to Netlify

1. Push this repo to GitHub
2. Connect to Netlify (Import from Git)
3. Add environment variable: `ANTHROPIC_API_KEY` = your key
4. Deploy!

## Features
- 🏠 Home dashboard with life state (🟢🟡🔴)
- 💬 Ada AI assistant (powered by Claude)
- 💳 Subscription tracking
- 📄 Document management
- ✅ Task management
- 🔄 Habit tracking
- 📅 Calendar view
- 🌍 Multi-country support
